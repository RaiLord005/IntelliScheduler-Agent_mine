#  IntelliScheduler Agent

A smart meeting scheduling agent designed to streamline appointment management. This system distinguishes between personal and organizational meetings, offers real-time slot availability checks, and prevents scheduling conflicts.

##  Key Features
* **Dual Scheduling Modes:** Toggle between "Person-to-Person" and "Organization-Based" meetings.
* **Smart Conflict Detection:** Automatically detects and blocks busy time slots (Red) to prevent errors.
* **Dashboard Filtering:** A unique feature to filter your meeting history list by type (Person vs Organization).
* **Local Persistence:** Data is saved locally in the browser, ensuring meetings aren't lost on refresh.

##  Tech Stack
* **Frontend:** HTML5, CSS3 (Glassmorphism UI), JavaScript (ES6+)
* **Data Handling:** LocalStorage & JSON
* **Architecture:** Vanilla JS (Client-Side)

##  How to Run
1. Clone the repository.
2. Navigate to the `templates` folder.
3. Open `index.html` in your browser.
